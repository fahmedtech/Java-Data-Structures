  1. public void append(E e) -- Appends the specified/parameter element to the end of this list.  If the parameter is null, the method throws a NullPointerException (and returns).  After the successful insertion, the method also increments N (the number of elements in the list).  Be careful when the list is empty.
  2.  public boolean contains(E e) -- Returns true if this list contains the specified element, or false otherwise.  If the parameter is null, the method throws a NullPointerException (and returns).  The comparison was done using equals().
  3. public E get(int k) -- Returns the element at the specified position in this list.  The position (k) is a 0-based index.  For example, if the list contained  [4, 8, 2, 7] and get(1) is called in this list, the method returns the element 8.  The method also throws an IndexOutOfBoundsException if the index is out of range (k < 0 || k >= size()).
  4. public void removeAt(int k) -- Removes the element at the specified position in this list (and decrements N).  The position (k) is a 0-based index.  For example, if the list contained  [4, 8, 2, 7] and remove(1) is called, the method changes the list to [4, 2, 7].  The method also throws an IndexOutOfBoundsException if the index is out of range (k < 0 || k >= size()).
  5.  public void clear() -- Sets head to null and N to 0.
